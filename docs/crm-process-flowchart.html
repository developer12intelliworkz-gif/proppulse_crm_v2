Enter the basic details of the new real-estate project

👤 Sales Manager / Admin 📺 Screen: Projects → Create Project → Step 1

Sales Manager or Admin creates the project record so the team can start adding details.

After saving, you move to Step 2 — Add Address in the project wizard.

The system saves the project name, dates, sales assignment, and settings. The project is created as a draft and you are taken to the Address step. Notification emails are sent if you added any.

Finish Project Profile

Connect to property portals and mark the project as complete

👤 Sales Manager / Admin 📺 Screen: Create Project → Step 6 (Portal Integration)

Who does this?

Admin or Sales Manager finalizes the project for listing on external portals.

What triggers the next step?

Once marked Complete, the project is ready for Step 4 — Project Setup (inventory configuration).

🌐 Portal Integration Form
Aa
India Property Code 🟢 One required
Your listing code on India Property portal
Aa
Magicbricks Code 🟢 One required
Your listing code on Magicbricks portal
Aa

✅ When you click Finish / Submit

The system checks that name, address, at least one specification, at least one brochure, and a portal code are all filled in. If everything is OK, the project is marked Complete and saved. You can now set up inventory (towers, units, etc.) in Project Setup.

Project profile complete → Go to Project Setup

👤 Admin / Inventory Manager 📺 Screen: Project Setup

Who does this?

Admin or inventory manager configures how the project's units are organized (towers, sectors, plots, etc.).

What triggers the next step?

After creating at least one tower/sector/phase, you can Add Units under each section.

✅ When you click Generate Quotation

The system calculates: Total Area = Carpet + Super Built-up, then Basic Price = Total Area × Price per sq ft. It adds each charge line (GST, PLC, etc.) in order and creates a unique quotation number. A PDF with your company letterhead is ready to download and share with the client. The quote is saved permanently — it cannot be edited after creation.

After the quotation is shared with the client…

✅ Accepted Only one accepted quote allowed per unit. Unit pricing is confirmed. Proceed with booking.
❌ Rejected Client declined. You can generate a new quotation with different options or pricing.
📝 Draft / Sent Quote saved or shared but not yet decided. Follow up with the client.

Scope of Improvement

Suggested enhancements to simplify the CRM for sales teams. This guide reflects the recommended experience — some items below are not yet live in the app.

  • 1
    Remove "Is Active" from Create Project (Step 1) New projects do not need an active/inactive toggle at creation time. The system should default all new projects to active. This field adds unnecessary confusion for sales staff on day one and is better managed later from Project Management if a project must be hidden.
  • 2
    Simplify the 6-step project wizard Combine Address and Office Address into one screen, and group VR + Amenities into a single "Marketing" step to reduce clicks before a project is ready for inventory setup.
  • 3
    Auto-enable Inventory when Project Setup starts Instead of asking users to tick "Enable Inventory" during project creation, automatically turn on inventory when they open Project Setup — with a clear one-time confirmation.
  • 4
    Quotation status updates from the UI Allow sales executives to mark a quotation as Sent, Accepted, or Rejected directly from the quotation list — without needing admin support.
  • 5
    Pre-fill client name from assigned lead When generating a quotation for a unit that already has a lead assigned, automatically fill the client name and show lead contact details on the PDF.
  • 6
    Clearer duplicate unit warning Show the existing unit's tower/sector when a duplicate unit number is entered, so sales staff know exactly which record conflicts.
